# Vignesh K Portfolio

A modern, interactive portfolio built with React, Three.js, and Vite.

## Features

- ⚡ **Fast & Modern Stack**: React 19 + Vite + TypeScript
- 🎨 **3D Graphics**: Powered by Three.js with Framer Motion animations
- 📱 **Responsive Design**: Tailwind CSS for beautiful UI
- 🎭 **Interactive Elements**: React Three Fiber for immersive experience

## 🚀 Live Demo

Visit: [https://itsmevicky.github.io/portfolio2/](https://itsmevicky.github.io/portfolio2/)

## 📦 Tech Stack

- **Frontend**: React 19, TypeScript
- **3D**: Three.js, React Three Fiber, Drei
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React, React Icons
- **Build**: Vite
- **Routing**: React Router

## ⚙️ Setup & Development

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Opens at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

Output in `dist/` directory

### Preview Production Build

```bash
npm run preview
```

## 📁 Project Structure

```
src/
  ├── components/     React components
  ├── pages/          Page components
  ├── App.tsx         Main app component
  └── main.tsx        Entry point
public/               Static assets
dist/                 Production build (auto-generated)
```

## 🌐 Deployment

This portfolio is set up to deploy to GitHub Pages. To enable:

1. Go to **Settings → Pages**
2. Set source to **Deploy from a branch**
3. Select branch: **main** and folder: **/ (root)**

The site will be available at: `https://itsmevicky.github.io/portfolio2/`
